## React Practice Project (Vite + Tailwind CSS)

This is a practice project built using React, Vite, and Tailwind CSS.  
It focuses on core React concepts like components and state management.

### Features
- Counter App using useState hook
- Simple eCommerce UI design using Tailwind CSS
- Popup/Modal enquiry form
  - Opens on button click
  - Closes with close button
- Component-based architecture
- State management using React hooks

### Tech Stack
- React
- Vite
- Tailwind CSS
- JavaScript

### Purpose
This project was built for practicing React fundamentals including components, props, and state management using hooks.

### How to Run
npm install
npm run dev
